# False MLBB Overlay Tool V4.8 

A professional broadcast overlay and automation tool for Mobile Legends: Bang Bang (MLBB), designed to provide dynamic hero drafting, match statistics, and team branding layouts for esports tournaments.

This repository includes custom modifications that introduce an **MPL/BOG Hybrid Style Layout**, offering a cleaner and more professional broadcasting experience.

## 🚀 Features
- **Dynamic Drafting UI**: Real-time hero pick and ban animations.
- **Clean Header Layout**: Minimalist placement for Coach icons and names at the top edges.
- **Center Console (BOG Style)**: Centralized area for Match Info, Map Details, and animated Score Bars (in progress).
- **Smart Text Handling**: Automatically wraps and scales long team or school names to ensure the UI remains pristine without overlapping hero cards.
- **Web-based Control Panel**: Manage the entire broadcast overlay seamlessly from a browser.

## ⚙️ Setup & Installation
1. Ensure you have [Node.js](https://nodejs.org/) installed.
2. Clone this repository.
3. Open the folder in your terminal and run `npm install` to install the necessary dependencies.
4. Start the tool by running the provided `FALSEMLBBOVERLAYTOOL.BAT` or `FALSEMLBBOVELAYFIX.bat` file (or run `node server.js` manually).
5. Open the provided `localhost` links in your browser to access the Control Panel, and add the Display links as Browser Sources in OBS Studio/vMix.
